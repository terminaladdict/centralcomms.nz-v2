We set up a complete CCTV system with 6 cameras, cabling, and an NVR server at an address in Ellicott Road, Hamilton. The system uses a combination of dome and bullet cameras.

We're proud to have used completely open-source NVR software — this allows us to do things like embed live camera streams in a website.

We wrote custom code and embedded a number of cameras in a website that integrated with other information on the property. We also integrated an Android application to view the stream directly from each camera, and another Android app to view alarms and recordings from the NVR.

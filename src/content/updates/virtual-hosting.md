Central Communications is a huge fan of open source technology. When developing a cluster of virtual servers, we investigated all the options and chose the open source platform Proxmox. Proxmox allows us to cluster real machines together. These real machines host virtual machines that we can migrate in real time between hosts. Proxmox VE is a complete open-source platform for enterprise virtualisation — managing VMs, containers, software-defined storage and networking, and high-availability clustering from a single web interface.
